# v1.0 polish audit

Findings + fixes-applied for the five polish sub-tasks listed in
the v1.0 plan (PR 9: 11a–11e).

## 11a — Citations consolidation

**State:** Citations are already centralized in `lib/citations.ts`.
About's master list reads from this file. Faust, Dante, Bridges
sections, and Depth Psychology essays cite by string key matching
entries in `citations.ts`.

**Findings:**

- The citation list grew significantly during the Faust + Dante
  builds (Friston, Berridge & Robinson, Mar, Speer, Wallentin,
  Panksepp). All have verifiable DOIs or canonical publisher
  citations.
- One earlier duplicate (`panksepp-1998-affective-neuroscience`)
  was caught during the Faust commit and resolved with a cross-
  reference comment in `citations.ts`.

**Action:** No additional consolidation needed for v1.0. The
single-source-of-truth pattern is already in place.

## 11b — Image provenance

**State:** Every historical plate, mandala, and figure has author
+ title + year attribution at the call site or in the page's
sources block. Plates live in `public/mandalas/` and `public/plates/`
under license-clear names (Fludd, Hildegard, Bhavachakra,
Sri Yantra, Aztec Sun Stone, Chartres Rose, Splendor Solis,
Delacroix Faust, Blake Dante).

**Findings:**

- All images on the site are public-domain (pre-1928 publications,
  late-Renaissance / medieval works, or Wikimedia-Commons-confirmed
  PD scans).
- The Archetypes carousel renders attribution under each mandala
  via the `MandalaCard` component's caption block.
- The literary rooms' Delacroix + Blake plates carry inline
  attribution via the `PassageAnalysis` component.

**Action:** No changes needed for v1.0.

## 11c — Translation attribution

**State:** Both shipped literary rooms credit their English
translations:

- **Faust:** Bayard Taylor, 1870 (public domain). Cited at the
  passage call site.
- **Dante:** Henry Wadsworth Longfellow, 1867 (public domain).
  Cited at the passage call site.

**Findings:**

- `grep` for "Mandelbaum" returns no matches. The original draft
  mentioned a possible Mandelbaum citation; the shipped pages use
  Longfellow exclusively.
- `grep` for "Wayne" / "Hollander" / other Dante translators —
  no matches.

**Action:** No changes needed for v1.0.

## 11d — Footer consistency

**State (before):** The home page (`app/[locale]/page.tsx`) had
an inline three-caption footer (built / encoder / about). Every
other page had no footer at all — pages just ended at their last
section.

**Action (this PR):** Extracted a shared `<SiteFooter>` component
mounted in `app/[locale]/layout.tsx`. Every page now ends with
the same four-row footer:

1. 8 room links (Mirror · Music · Cross-Cultural · Cellular ·
   Threshold · Archetypes · Faust · Dante)
2. 7 reference links (Atlas · Bridges · Tours · Depth Psychology ·
   Field Notes · Site Map · About)
3. "Built at Chulalongkorn JIPP · TRIBE v2 encoder"
4. "v1.0 · last reviewed 2026-05-14 · Source on GitHub"

The home page's old inline footer is removed (the shared one
takes its place).

i18n: new `footer.{built,encoder,version,github}` namespace
across all 6 locales. Room labels reuse the existing flat
`nav.<key>` namespace.

## 11e — Mandala rotation decision

**State:** Two of the seven Archetypes mandalas appear elsewhere
on the site:

- **Fludd microcosm** — used as the decorative background on
  `/field-notes` index and `/map` index.
- **Hildegard codex** — used as the decorative background on
  `/threshold`.

**Findings:** These recurrences are intentional. The mandalas
are decorative recurring motifs, not unique-per-page assets. The
microcosm-on-map placement is the page's argument (an index is
a microcosm of the whole). Threshold's Hildegard is similarly
motif-coupled.

**Action:** No changes needed for v1.0. The recurrence pattern
is consistent with the editorial intent.
